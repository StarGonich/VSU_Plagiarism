1) Python (у меня используется 3.12)
Следующие два шага можно пропустить, так как для тестов я уже создал графы в директории `test_graphs` и залил вместе с иходниками
2) Установка Joern 4.0.xxx версии
3) Запуск sh файла, где в качестве параметра используется директория с программами
```bash
.\files_to_graph.sh .\-1\
```
В итоге появится несколько папок, но нас в основном интересует test_graphs, где находятся PDG графы наших программ для теста.

4) Запуск main.py, где берутся графы из директории `test_graphs` и сравниваются. По итогу создаётся директория `test_logs`, где хранится выводы
5) Запуск gui.py, где можно в графическом интерфейсе посмотреть на результаты.
